Authentication

POST api/auth/register

Description: Register new user

Headers:

  • Content-Type | application/json
  • X-Requested-With | XMLHttpRequest

Example of Body in JSON for request:

{ "username":"Tester", "email": "tester@mail.com", "password":"secret", "password_confirmation":"secret" }

POST api/auth/login

Description: Authorization to get access

Headers:

  • Content-Type | application/json
  • X-Requested-With | XMLHttpRequest

Example of Body in JSON for request:

{ "email":"admin@mail.com", "password":"admin", "remember_me":true }

Response: access_token, token_type, expires_at

Users

GET api/users

Description: Returns list of users

Headers:

  • Authorization | Bearer $Access_Token

GET api/users/{user_id}

Description: Returns single user

Headers:

  • Authorization | Bearer $Access_Token

PUT api/users/{user_id}

Description: Update a user

Headers:

  • Authorization | Bearer $Access_Token

Example of Body in JSON for request:

{ "username": "tester", "first_name": "Tester", "last_name": "Testerson", "email":"tester@mail.com", "password":"secret", "password_confirmation":"secret" }

DELETE api/users/{user_id}

Description: Delete user

Headers:

  • Authorization | Bearer $Access_Token

POST api/users/add-user-to-company

Description: Adds user to company

Headers:

  • Authorization | Bearer $Access_Token

Example of Body in JSON for request:

{ "user_id": 2, "company_id": 3 }

PUT api/users/update-user-company/{id}

Description: Updates user in company

Headers:

  • Authorization | Bearer $Access_Token

Example of Body in JSON for request:

{ "user_id": 2, "company_id": 3 }
