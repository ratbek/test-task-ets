Инструкция по развертыванию и запуску

Шаг 1. Склонировать репозиторий с https://github.com/ratbek/test-task-ets.git

Шаг 2. Создать и настроить virtual-host для nginx. Файл с настройками (test-task.conf) находится в корневой папке.

Шаг 3. Настроить подключение к базе данных в файле .env.

Шаг 4. Импортировать базу данных. Для этого в корневой папке имеется sql dump файл (test_task_db.sql).
        Если этот шаг выполнен, то следующие два шага не нужно выполнять.

Шаг 5. (Опционально) Выполнить миграцию.
		php artisan migrate

Шаг 6. (Опционально). Если нужны какие-то данные в базе, то можно заполнить данными с помощью Seeder`ов.
		Для это надо выполнить следующие команду последовательно:
		php artisan db:seed --class=CompaniesTableSeeder
		php artisan db:seed --class=UsersTableSeeder
		php artisan db:seed --class=CompanyUserTableSeeder

Шаг 7. Теперь можно запускать тестовое задание (далее сайт) в браузере.

Шаг 8. Если запустилось, то можно проверять с помощью Postman, так как на сайте полностью реализован REST API.
        В корневой папке есть файл API-reference-index.html, где описано детали взаимодействия с сайтом.
